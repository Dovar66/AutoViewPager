# AutoViewPager
一个实现自动轮播并且可以无限轮播的viewpager，可修改滑动方向为纵向滑动

Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
Step 2. Add the dependency

	dependencies {
	        compile 'com.github.Dovar66:AutoViewPager:1.0'
	}
